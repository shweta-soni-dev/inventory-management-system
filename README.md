# Inventory Management System (Java)

A Java-based Inventory Management System built for job preparation.  
It helps manage products and stock records in a simple way.

## Features
- Add new products
- Update product details / stock
- Delete products
- View product list
- Search products

## Tech Stack
- Java
- (MySQL + JDBC)  *(if your project uses database)*
- NetBeans / Eclipse (any IDE)

## How to Run
1. Download / clone this repository
2. Open the project in NetBeans (or your IDE)
3. If using MySQL:
   - Create a database (example: `inventory_db`)
   - Import your SQL file (if available)
   - Update DB username/password in code
4. Run the Main class

## Project Structure
- `src/` : source code
- `lib/` : required libraries
- `dist/` : built output (if present)
